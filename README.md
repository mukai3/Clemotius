# Clemotius (クレマチウス)

Windows 向けのマウスジェスチャー+αのユーティリティです。更新が終了してしまっている超有名マウスジェスチャーソフト[^1]にインスパイアされて、Windows 10/11 向けに **.NET (C#) で再実装**した個人用プロジェクトです。


[^1]: 日本に自生するクレマチスの原種のひとつ「カザグルマ」

## 主な機能

- **マウスジェスチャー** — 右ボタンドラッグの軌跡（例: `←` 戻る、`→` 進む、`↓→` タブを閉じる）にアクションを割り当て。
- **右ボタン + ホイール** — 右ボタン押下中のホイール回転にアクションを割り当て（例: タブ切替）。
- **アプリ別プロファイル** — 前面アプリのプロセス名でジェスチャー定義を切り替え。プロファイルに一致したアプリでのみジェスチャーが有効になり、未登録のアプリは右ボタンをそのまま透過する（エクスプローラ等の右ドラッグやアプリ独自ジェスチャーをそのまま使える）。
- **拡張スクロール** — 修飾キー(Shift/Ctrl/Alt とその組合せ)＋ホイールでの動作、スクロールバー上でのホイール動作。Chrome/Edge などの **Chromium 系カスタムスクロールバー**にも対応。
- **タイトルバーアクション** — タイトルバーや最小化/閉じるボタンへの各種クリックに、常に最前面・ウィンドウシェード（巻き上げ）・実行ファイルのフォルダを開く・半透明化 を割り当て。
- **タスクトレイ常駐** — 設定画面の呼び出し、一時停止など。

アクションは「キー送信」「アプリケーションコマンド(WM_APPCOMMAND)」「ウィンドウを閉じる」などから選べ、よく使うショートカットはプリセットとして用意しています。

## 動作要件

- Windows 10 / 11 (x64)
- [.NET 10 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/10.0)（配布物はフレームワーク依存のため必要）

## インストール / 実行

1. [Releases](https://github.com/mukai3/Clemotius/releases) から `Clemotius-vX.Y-win-x64.zip` をダウンロード。
2. 任意のフォルダに展開し、`Clemotius.exe` を起動。
3. タスクトレイのアイコンから設定画面を開いて割り当てを編集。

設定は `%APPDATA%\Clemotius\config.json` に保存され、手動編集も即時反映されます（破損時は `.bak` に退避して既定設定で起動）。

## 設定画面

WPF + [WPF-UI](https://github.com/lepoco/wpfui) による Fluent デザインの設定画面（左サイドナビ）。変更は即時適用されます。

| ページ | 内容 |
|---|---|
| ジェスチャー | アプリ別プロファイル、ジェスチャー一覧、右ボタン+ホイール |
| 拡張スクロール | 修飾キー6種＋ホイールの動作 |
| ホイール | スクロールバー上でのホイール動作 |
| ウィンドウ | タイトルバーアクション、半透明化の不透明度 |
| 詳細 | トレイ表示、ジェスチャー判定（距離/タイムアウト/長押し）、軌跡・コマンド表示 |

## 開発 / ビルド

```powershell
# ビルド
dotnet build Clemotius.slnx -c Release

# テスト
dotnet test Clemotius.slnx

# 配布物（フレームワーク依存・単一フォルダ）
dotnet publish src/Clemotius/Clemotius.csproj -c Release -r win-x64 --self-contained false -o publish
```

- 構成
  - `src/Clemotius` — WinForms ホスト常駐 + WPF 設定画面 + 低レベルフック等の副作用層
  - `src/Clemotius.Core` — Win32 非依存のロジック（設定モデル、ジェスチャー判定、スクロール解決など。ユニットテスト対象）
  - `tests/Clemotius.Tests` — xUnit テスト
- ターゲット: `net10.0-windows`（WPF + WinForms）
- 入力捕捉は低レベルフック（`WH_MOUSE_LL` / `WH_KEYBOARD_LL`）。DLL インジェクションは行いません。

# 独り言

WindowsやChromiumのアップデートがある度に少しずつ機能を失っていき、ちょっとした不具合にイラつきながら使い続けている某インスパイア元ツールから乗り換えるべく、今まで色々なマウスジェスチャーソフトを試してきましたが、完璧なものには未だに出会えていません。
あの機能さえあれば…この不具合さえなければ…そう考えながらClaudeCodeに相談していたらこんなものが完成しました。
某インスパイア元ツールユーザの目線から違和感がないように、それでいてモダンなUIを目指しています。

自分が使っていた機能を優先して実装していますが、「これがないから乗り換えられないんだよね」という機能があればこっそり教えてください。対応するかも。
